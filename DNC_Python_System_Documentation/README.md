# DNC参数计算系统 - 完整文档集

## 项目简介

DNC参数计算系统是一个专业的数控设备参数管理软件，用于从电脑直接传送参数到NC机的宏变量系统。本项目提供了从VB.NET到Python技术栈迁移的完整解决方案。
